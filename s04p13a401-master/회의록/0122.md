2. 게시판 글 상세보기 페이지 목업 (혜진)
3. 메인페이지 수정 (혜진)
    - 검색 부분 직관적으로 바꾸기 (통합, 오피셜, 커스텀 드랍다운으로 선택 가능하도록)
4. 검색 페이지 (세진)

5. 회원가입 (채은)
    - 이메일 중복확인 → 인증 메일 전송 후 코드 입력으로 수정 (어제 팀회의 이슈)

8. 관리자 시점 페이지 추가 (세진)
    - 게시글에 수정/삭제 보여지도록
9. 레시피 작성페이지 수정 (혜진)
